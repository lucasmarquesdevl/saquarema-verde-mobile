# 🌿 Saquarema Verde - Mobile App

Repositório destinado ao aplicativo mobile do projeto **Saquarema Verde**, desenvolvido com React Native (Expo).

## 📌 Descrição do Projeto
O aplicativo permite que os usuários visualizem os eventos, trilhas, praias e pontos turísticos cadastrados pela administração. Os usuários podem conferir as datas, descrições e assistir aos vídeos promocionais de cada local diretamente pelo celular.

## 📱 Tecnologias Utilizadas
- **React Native** (Framework principal)
- **Expo** (Ferramenta de desenvolvimento e build)
- **React Navigation** (Navegação em abas e pilhas)
- **Expo AV / Expo Video** (Reprodução de vídeos no app)
- **Fetch API** (Comunicação com o backend Node.js)

## ⚙️ Requisitos para Rodar o App
- Aplicativo **Expo Go** instalado no seu celular físico (Android/iOS).
- Computador e Celular conectados na **mesma rede Wi-Fi**.
- O Backend (servidor Node.js + MySQL) precisa estar rodando localmente na sua máquina.

## 🚀 Como Executar Localmente
1. Certifique-se de que o backend já está rodando (na porta 8080).
2. Verifique o seu IP local (ex: `192.168.1.x`) e confira se ele está configurado corretamente no arquivo `src/services/api.js`.
3. Abra este repositório no terminal.
4. Instale as dependências:
   ```bash
   npm install
   ```
5. Inicie o aplicativo:
   ```bash
   npm start
   ```
   *ou*
   ```bash
   npx expo start
   ```
6. Abra o aplicativo da câmera no seu celular e escaneie o **QR Code** que aparecer na tela do terminal ou na aba do navegador.
